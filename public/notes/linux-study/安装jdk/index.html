yum安装jdk

yum install java-1.8.0-openjdk* -y

手动安装

下载好jdk-8u112-linux-x64.tar.gz 提取码：liak

下载好Xshell 6和Xftp 6提取码：9b13rvice

上传安装包

使用Xftp连接云服务器，连接可能会出现"无法与XXX连接”，别慌

执行以下命令，观察22号端口有没有被监听

netstat  -anp  |  more

没有启动的话，重启一下

service  sshd  restart

安装

打开Xshell连接到服务器跳转到/opt目录下

cd  /opt

解压opt文件夹下的jdk-8u112-linux-x64.tar.gz文件

tar  -zxvf jdk-8u112-linux-x64.tar.gz

配置jdk环境变量

编辑/etc/目录下的profile文件

vim    /etc/profile

按下 Esc键 进入命令行模式 输入G 进入最后一行 按下 i 键进入编辑模式 ，输入以下配置

JAVA_HOME=/opt/jdk1.8.0_112
PATH=/opt/jdk1.8.0_112/bin:$PATH
export  JAVA_HOME PATH

按下 Esc键 进入命令行模式，输入 wq! 强制退出保存，再logout注销命令

重新登录后 输入java 输入javac 说明安装成功

编写Hello

vim  /opt/Hello.java

按下 i 键进入编辑模式 ，输入以下配置

public class Hello{
	public static void main (String[] args){
		System.out.println("Hellow");
}

按下 Esc键 进入命令行模式，输入 wq! 强制退出保存输入

javac Hello.java
