配置阿里云镜像加速地址

注册阿里云账号

https://cr.console.aliyun.com/cn-hangzhou/instances/mirrors

复制加速地址

编辑配置文件

vim /etc/docker/daemon.json

添加如下内容

{/
  "registry-mirrors": ["https://****.mirror.aliyuncs.com"]
}

重启docker服务

systemctl daemon-reload

systemctl restart docker

检查

docker info

配置成功！！！！
