配置Tomcat

拉取Tomcat镜像

docker pull tomcat

运行tomcat

将宿主机的8081端口指向到容器的8080端口

docker run -d --name tomcat -p 8081:8080 tomcat

发现Tomcat已经启动了但是访问不了

进入交互模式发现webapps是空的，资源都在webapps.dist下

docker exec -it 926d769607f4 /bin/bash

将webapps.dist下的所有文件复制到webapps

cp -R webapps.dist/*  webapps

再次访问成功！！！！！
