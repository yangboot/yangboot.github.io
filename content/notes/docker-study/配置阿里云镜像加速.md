### 注册阿里云账号

https://cr.console.aliyun.com/cn-hangzhou/instances/mirrors

复制加速地址

![](/img/docker/1.JPG)

### 编辑配置文件

```shell
vim /etc/docker/daemon.json
```
添加如下内容
```json
{/
  "registry-mirrors": ["https://****.mirror.aliyuncs.com"]
}
```
### 重启docker服务
```shell
systemctl daemon-reload
```
```shell
systemctl restart docker
```
### 检查

```shell
docker info
```
![](/img/docker/2.jpg)
配置成功！！！！
